Hexo搭建Github博客

安装node

到http://nodejs.cn/下载最新版本安装包，然后安装即可。

安装git bash

到https://git-scm.com/downloads下载安装包，然后安装即可。主要是用于将本地的hexo内容提交到github上面去。

安装hexo

本地创建一个空文件夹，名字可以自定义，我这边用blog_hexo，在这个里面存放hexo的配置文件，到blog_hexo目录下安装hexo
npm install -g hexo

初始化hexo

hexo init

安装一个完整项目

hexo server (简写： hexo s)

现在本地的博客已经创建好了，项目目录如下

在浏览器上打开：localhost:4000
会看到一个默认的页面，如果不喜欢这个默认的主题，可以找自己喜欢的主题来替换，https://hexo.io/themes/官方网站有很多主题可以选择，我自己用的主题是next

安装next主题

在命令行中执行：
git clone https://github.com/iissnan/hexo-theme-next.git themes/next
执行完毕后，修改blog_hexo根目录下的 _config.yml 文件中 theme: next

更新主题

cd themes/next
git pull

完成后效果如下

此时，本地的博客项目已经搭建完成，接下来就是把它部署到github上面

部署github

首先到https://github.com官网注册一个属于自己的账号，然后创建git仓库，如下图

仓库名最好和用户名保持一致，因为我已经创建过了，所以标红，大家可以忽略

本地博客关联自己的github账号

首先复制自己的git地址，如下图

再次打开根目录下的 _config.yml 文件，在最底部修改配置

修改deploy配置：

deploy:
type: git
repo: https://github.com/liyina91/liyina91.github.io.git
branch: master

上传博客

首先需要执行下列命令：

npm install hexo-deployer-git --save

安装完成后，执行命令：

hexo deploy (简写 hexo d)

然后浏览器中输入 https://自己的用户名.github.io/ 便可以访问了

命令集合

博客写好之后放在source/_post目录下，执行命令

hexo clean （可用简写 hexo c）
hexo generate （可用简写 hexo g）
hexo deploy （可用简写 hexo d）
