Hexo搭建Github博客

安装node

到 http://nodejs.cn/ 下载最新版本安装包，然后安装即可。

安装git bash

到 https://git-scm.com/downloads 下载安装包，然后安装即可。主要是用于将本地的hexo内容提交到github上面去。

安装hexo

本地创建一个空文件夹，名字可以自定义，我这边用blog_hexo，在这个里面存放hexo的配置文件，到blog_hexo目录下安装hexo
npm install -g hexo

vue数据双向绑定原理

vue实现数据双向绑定是采用数据劫持结合发布者-订阅者模式的方式，通过Object.defineProperty()来劫持各个属性的setter，getter，在数据变动时发布消息给订阅者，触发相应的监听回调。
